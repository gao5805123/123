给指定路径配置Path环境变量,实现路径下的文件夹、文件和程序的快速打开
发表于 2020-05-07| 更新于 2020-05-17|Windows
阅读量:

作为一个程序员，需要使用的软件非常多，桌面总是凌乱不堪，找东西总是很烦恼；使用一些桌面管理工具虽然有美观的，但是有些时候程序会突然崩溃，致使桌面又回到了从前的模样，必须依靠重启来恢复

在桌面创建一个文件夹

文件夹起名为程序快捷键(可以按个人意愿命名)，将所有的程序快捷键放入其中(程序的快捷键建议改一下,改成方便记忆的名称，如：腾讯QQ改成QQ，把自动生成的IDEA快捷键名改成IDEA)

为该文件夹配置环境变量

右键我的电脑，点击属性；找到并点击高级系统设置

点击环境变量

找到系统变量下的Path环境变量，点击编辑

点击新建，再点击浏览，找到在桌面创建的文件，点击确定

点击所有的确定完成配置

使用运行窗口打开程序

Win + R打开运行窗口,输入QQ(英文不区分大小写，程序名必须一致)，等待程序启动
