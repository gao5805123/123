Windows去除图标小箭头
发表于 2020-05-17| 更新于 2020-06-13|Windows
阅读量:

方法一

Win + R键打开运行窗口，输入regedit，打开注册表编辑器

右键HKET_CLASSES_ROOT，点击查找

输入lnkfile，勾选查看下的项并勾选全字匹配(W)，点击查找下一个，等待查找完成

查找完成后会自动跳到lnkfile目录下，将IsShortcut右键删除(无需在意弹出的警告信息)，重启电脑即可

若想让图标上的小箭头重新显示，在Inkfile下右键新建字符串值(S)

将其名称改为IsShortcut，重启电脑即可
