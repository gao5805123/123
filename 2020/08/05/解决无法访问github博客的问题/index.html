解决无法访问github博客的问题
发表于 2020-08-05| 更新于 2020-10-24|博客
阅读量:

异常原因

解决方法

IP查询

Win + E打开我的电脑，用记事本打开C盘下的Windows下的System32下的drivers下的etc下的hosts文件，将IP地址和域名都粘贴到当中保存

Win + R打开运行框，输入cmd打开命令提示符窗口，输入该命令刷新DNS解析缓存后再次访问即可
